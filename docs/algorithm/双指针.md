# 双指针

### 15.三数之和

给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：**答案中不可以包含重复的三元组。

思路：第一层循环只循环i，然后让l放在i的右侧一个，r是末端，然后遍历，如果三数加起来大于0，那么就让r--,小于0那就让l++,等于零的时候就把这组push到答案里，然后循环，直到l和r都是和上一个不同的数，！！！注意去重：i拿来判重，保证下一个nums[i]不和上一个相等即可

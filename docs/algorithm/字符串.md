### 14.最长公共前缀

给定字符串数组 `strs`，查找其中的最长公共前缀。如果不存在公共前缀，返回空字符串 `""`。

- **思路**：先找到数组里最短字符串的长度 `m_len`，最长公共前缀不可能超过它。然后按列扫描下标 `i`，取第一串的第 `i` 个字符 `c`，检查所有字符串在该位置是否都等于 `c`，若有一个不等则前缀结束；否则把 `c` 追加到答案中，继续下一位
- **代码**：
  ```c++
  class Solution {
  public:
      string longestCommonPrefix(vector<string>& strs) {
          int n = strs.size();
          int m_len = INT_MAX;
          string ans;
          for (int i = 0; i < n; i++)
              m_len = min(m_len, (int)strs[i].size());
          for (int i = 0; i < m_len; i++) {
              char c = strs[0][i];
              bool ok = true;
              for (int j = 0; j < n; j++) {
                  if (strs[j][i] != c) {
                      ok = false;
                      break;
                  }
              }
              if (ok) ans += c;
              else break;
          }
          return ans;
      }
  };
  ```

### 151.反转字符串中的单词

给定字符串 `s`，反转其中单词顺序。单词由非空格字符组成，单词间用单个空格连接。需处理前导、尾随空格及单词间多余空格，返回结果中单词间仅一个空格、无额外空格。

- **思路**：按字符遍历，遇到非空格就拼到当前 `word`，遇到空格且 `word` 非空就把 `word` 压栈并清空。遍历完后若 `word` 还有内容再压栈。依次弹栈拼到答案，每弹一个单词加一个空格，最后去掉末尾多余空格。
- **代码**：
  ```c++
  class Solution {
  public:
      string reverseWords(string s) {
          stack<string> st;
          string word, ans;
          for (int i = 0; i < (int)s.size(); i++) {
              if (s[i] == ' ' && !word.empty()) {
                  st.push(word);
                  word.clear();
              }
              if (s[i] != ' ') word += s[i];
          }
          if (!word.empty()) st.push(word);
          while (!st.empty()) {
              ans += st.top();
              st.pop();
              ans += " ";
          }
          if (!ans.empty()) ans.pop_back();
          return ans;
      }
  };
  ```
- 题解：`code/151.反转字符串中的单词.cpp`

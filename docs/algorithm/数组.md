# 数组

## 经典题目

### 两数之和

- 思路：哈希表存已遍历的值，查找 complement
- 题解：`code/两数之和.js` 或 `code/twoSum.js`

### 三数之和

- 思路：排序 + 双指针
- 题解：`code/三数之和.js` 或 `code/threeSum.js`

### 最大子序和

- 思路：动态规划，dp[i] 表示以 nums[i] 结尾的最大和
- 题解：`code/最大子序和.js`

### 数组扁平化

- 思路：递归或 reduce 实现 flat
- 题解：`code/数组扁平化.js` 或 `code/flat.js`

### 283.移动零

给定一个数组 `nums`，将所有 `0` 移动到数组的末尾，同时保持非零元素相对顺序不变，并且要求原地操作。

- 思路：遍历数组，遇到 `0` 记录其下标；遇到非零时，如果之前有 `0`，就把当前非零与最早的 `0` 交换，并把当前下标继续作为新的 `0` 位置

### 48.旋转图像

给定一个 `n x n` 的二维矩阵 `matrix` 表示一个图像，请你将图像顺时针旋转 90 度。

- **思路**：用一个同尺寸的临时矩阵 `tmp` 保存旋转后的结果：原位置 `(i, j)` 旋转后变为 `(j, n-1-i)`，最后再赋值回原矩阵 `matrix`。这是最直观的一种写法。

- **代码**：

  ```c++
  class Solution {
  public:
      void rotate(vector<vector<int>>& a) {
          int n = a.size();
          vector<vector<int>> tmp(n, vector<int>(n));
          for (int i = 0; i < n; i++) {
              for (int j = 0; j < n; j++) {
                  tmp[j][n - 1 - i] = a[i][j];
              }
          }
          a = tmp;
      }
  };
  ```

- 题解：`code/48.旋转图像.cpp`

### 54.螺旋矩阵

给你一个 `m` 行 `n` 列的矩阵 `matrix` ，请按照 **顺时针螺旋顺序** ，返回矩阵中的所有元素。

![螺旋矩阵示例](/images/p1.png)

输入：matrix = [[1,2,3],[4,5,6],[7,8,9]]
输出：[1,2,3,6,9,8,7,4,5]

- 思路：主要是理清楚边界的变化，设置一个空数组来放答案，如果数组的大小和矩阵的大小一样，则说明读取结束；主函数部分用一个while循环，里面依次使用四个for循环，分别是向右，向下，向左，向上。注意处理细节(当从左往右走完，那么上边界就应该++；如果从上往下走完，那么右边界应该--…)。

- 代码：

  ```c++
  class Solution {
  public:
      vector<int> spiralOrder(vector<vector<int>>& matrix) {
          int n=matrix.size();
          int m=matrix[0].size();
          vector<int> ans;
          int l_b=0;
          int r_b=m-1;
          int u_b=0;
          int d_b=n-1;
          int x=0,y=0;
          while(ans.size()<m*n){
              for(int i=l_b;i<=r_b;i++){
                  if (ans.size() >= m * n)
                      return ans;
                  ans.push_back(matrix[u_b][i]);
              }
              u_b++;
              for(int i=u_b;i<=d_b;i++){
                  if (ans.size() >= m * n)
                      return ans;
                  ans.push_back(matrix[i][r_b]);
              }
              r_b--;
              for (int i = r_b; i >= l_b; i--) {
                  if (ans.size() >= m * n)
                      return ans;
                  ans.push_back(matrix[d_b][i]);
              }
              d_b--;
              for (int i = d_b; i >= u_b; i--) {
                  if (ans.size() >= m * n)
                      return ans;
                  ans.push_back(matrix[i][l_b]);
              }
              l_b++;
          }
          return ans;
      }
  };
  ```

### 56.合并区间

以数组 `intervals` 表示若干个区间的集合，其中单个区间为 `intervals[i] = [starti, endi]` 。请你合并所有重叠的区间，并返回 *一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间* 。

- 思路：这是个典型的算法模板，先对每个区间按照首元素排序，分成两种情况，1）A段的尾元素比B段的首元素小 2）A段的尾元素比B段的首元素大。

  对于1）我们需要把A压进去

  对于2）我们需要把A的首元素作为新区间的首元素，把A,B中较大的尾元素作为新区间的尾元素即可

- 代码：

  ```c++
  class Solution {
      typedef pair<int,int> PII;
  public:
      vector<vector<int>> merge(vector<vector<int>>& a) {
          int n=a.size();
          vector<PII> arr;
          for(int i=0;i<n;i++){
              arr.push_back({a[i][0],a[i][1]});
          }
          sort(arr.begin(),arr.end());
          int i=1;
          vector<vector<int>> ans;
          int st=arr[0].first,ed=arr[0].second;
          while(i<n){
              int s=arr[i].first;
              int e=arr[i].second;
              //不连续的直接把上一段压进去
              if(s>ed){
                  ans.push_back({st,ed});
                  st=s;
                  ed=e;
              }else{   //连续的话，找右侧的最大值作为ed的值
                  ed=max(e,ed);  //这句是关键，
              }
              i++;
          }
          ans.push_back({st,ed});
          return ans;
      }
  };
  ```

  

### 59.螺旋矩阵II

给你一个正整数 `n` ，生成一个包含 `1` 到 $n^2$ 所有元素，且元素按顺时针顺序螺旋排列的 `n x n` 正方形矩阵 `matrix` 。

- 示例：

  ![螺旋矩阵II示例](/images/p2.png)

- 思路：和上面的螺旋矩阵思路非常类似，略

- 代码：

  ```c++
  class Solution {
  public:
      vector<vector<int>> generateMatrix(int n) {
          vector<vector<int>> a(n,vector<int>(n));
          int left=0;
          int right=n-1;
          int up=0;
          int down=n-1;
          int index=1;
          while(index<=n*n){
              for(int i=left;i<=right;i++){
                  a[up][i]=index++;
              }
              up++;
              for(int i=up;i<=down;i++){
                  a[i][right]=index++;
              }
              right--;
              for(int i=right;i>=left;i--){
                  a[down][i]=index++;
              }
              down--;
              for(int i=down;i>=up;i--){
                  a[i][left]=index++;
              }
              left++;
          }
          return a;
      }
  };
  ```

  

### 73.矩阵置零

给定一个 `m x n` 的矩阵，如果一个元素为 0 ，则将其所在行和列的所有元素都设为 0 。请使用 **原地** 算法。

- 示例 1：输入 `matrix = [[1,1,1],[1,0,1],[1,1,1]]`，输出 `[[1,0,1],[0,0,0],[1,0,1]]`
- 示例 2：输入 `matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]`，输出 `[[0,0,0,0],[0,4,5,0],[0,3,1,0]]`

- **思路**：
  - 解法一：用两个 set 记录要置零的行、列下标，两遍扫描先标记再置零。时间 O(m×n)，空间 O(m+n)。
  - 解法二（推荐）：用矩阵的**第一行**和**第一列**当标记位——若某行/列需要置零，就把对应 `m[i][0]` 或 `m[0][j]` 置 0。`(0,0)` 同时表示「第 0 行是否有 0」，所以第 0 列是否置零用单独变量 `col0`。第一遍扫描打标记，第二遍从**右下往左上**置零，避免覆盖第一行/第一列的标记。时间 O(m×n)，空间 O(1)。

- **代码**：

  ```c++
  class Solution {
  public:
      void setZeroes(vector<vector<int>>& m) {
          int n = m.size(), mm = m[0].size();
          bool col0 = false;

          for (int i = 0; i < n; i++) {
              if (m[i][0] == 0) col0 = true;
              for (int j = 1; j < mm; j++) {
                  if (m[i][j] == 0) {
                      m[i][0] = 0;
                      m[0][j] = 0;
                  }
              }
          }

          for (int i = n - 1; i >= 0; i--) {
              for (int j = mm - 1; j >= 1; j--) {
                  if (m[i][0] == 0 || m[0][j] == 0)
                      m[i][j] = 0;
              }
              if (col0) m[i][0] = 0;
          }
      }
  };
  ```

- 题解：`code/73.矩阵置零.cpp`

### 75.颜色分类

给定一个包含红色、白色和蓝色、共 `n` 个元素的数组 `nums` ，**[原地](https://baike.baidu.com/item/原地算法)** 对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

我们使用整数 `0`、 `1` 和 `2` 分别表示红色、白色和蓝色。

必须在不使用库内置的 sort 函数的情况下解决这个问题。

**思路**：直接冒泡排序就可以

**代码：**

```c++
class Solution {
public:
    void sortColors(vector<int>& nums) {
        //bubblesort
        int n=nums.size();
        for(int i=0;i<n;i++){
            for(int j=i+1;j<n;j++){
                if(nums[i]>nums[j]){
                    int t=nums[j];
                    nums[j]=nums[i];
                    nums[i]=t;
                }
            }
        }
    }
};
```

### 49.字母异位词分组

给定一个字符串数组 `strs`，将「字母异位词」组合在一起。可以按任意顺序返回结果。

- **思路**：对每个字符串排序，排序后的结果作为哈希表 `unordered_map<string, vector<string>>` 的 key，value 存放所有排序后相同的一组字符串。最后把哈希表中的所有 value 收集起来即可。

- **代码**：

  ```c++
  class Solution {
  public:
      vector<vector<string>> groupAnagrams(vector<string>& strs) {
          unordered_map<string, vector<string>> mp;
          for (string s : strs) {
              string key = s;
              sort(key.begin(), key.end());
              mp[key].push_back(s);
          }
          vector<vector<string>> ans;
          for (auto& entry : mp) ans.push_back(entry.second);
          return ans;
      }
  };
  ```

- 题解：`code/49.字母异位词分组.cpp`

### 168.Excel名称

给你一个整数 `columnNumber`，返回它在 Excel 表中对应的列名称。

- **思路**：本质是「26 进制」短除法，但没有 0，需要把余数为 0 的情况特殊处理成 `Z`，并且商减一。不断对 26 取余、入栈，最后从栈中依次弹出组成答案。

- **代码**：

  ```c++
  class Solution {
  public:
      string convertToTitle(int c) {
          stack<char> st;
          long long n = c;
          while (n > 26) {
              int t = n % 26;
              n /= 26;
              if (t == 0) {
                  st.push('Z');
                  n -= 1;
              } else {
                  st.push('A' + t - 1);
              }
          }
          st.push('A' + n - 1);
          string ans;
          while (!st.empty()) {
              ans += st.top();
              st.pop();
          }
          return ans;
      }
  };
  ```

- 题解：`code/168.Excel名称.cpp`

### 171.Excel列表序号II

给你一个字符串 `columnTitle` 表示 Excel 表格中的列名称，返回该列名称对应的列序号。

- **思路**：把每一位当成 26 进制的一位，从左到右遍历：`ans = ans * 26 + (当前字母 - 'A' + 1)`。

- **代码**：

  ```c++
  class Solution {
  public:
      int titleToNumber(string c) {
          long long ans = 0;
          for (char ch : c) {
              ans = ans * 26 + (ch - 'A' + 1);
          }
          return (int)ans;
      }
  };
  ```

- 题解：`code/171.Excel列表序号II.cpp`

### 215.数组第k个最大的元素

给定整数数组 `nums` 和整数 `k`，请返回数组中第 `k` 个最大的元素。

- **思路**：用大根堆。先堆化再进行堆排序，把最大的元素逐个放到末尾，最后数组位置 `n-k` 就是第 `k` 大。

- **代码**：

```c++
/*给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。

请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

你必须设计并实现时间复杂度为 O(n) 的算法解决此问题。
*/
class Solution {
    public:
        //堆化（大根堆，堆顶是max）
        //n->堆大小，i->要调整的结点下标
        void heapify(vector<int>& a,int n,int i){
            int largest=i;//假设父节点是最大值
            int left=2*i+1;
            int right=2*i+2;
            if(left<n&&a[left]>a[largest]){
                largest=left;
            }
            if(right<n&&a[right]>a[largest]){
                largest=right;
            }
            if(largest!=i){
                swap(a[i],a[largest]);
                heapify(a,n,largest);//重新调整被影响的堆
            }
        }
        int findKthLargest(vector<int>& a, int k) {
            //堆排序（nlogn）
            //数组首下标是0
            int n=a.size();
            //从后往前调整
            for(int i=n/2-1;i>=0;i--){
                heapify(a,n,i);
            }
            //取堆顶，进行排序
            for(int i=n-1;i>0;i--){
                swap(a[0],a[i]);//最大值放到末尾
                heapify(a,i,0);
            }
            return a[n-k];
        }
```

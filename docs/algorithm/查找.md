# 查找

## 二分查找

### 算法思路

1）二分寻找某个数

```c++
int binarySearch(vector<int>&nums,int target){
    int l=0;
    int r=nums.size()-1;
    while(l<=r){
        int mid=l+(r-l)/2;
        if(nums[mid]==target){
            return mid;
        }else if(nums[mid]>target){
            r=mid-1;// target在左边，排除mid
        }else{
            l=mid+1;// target在右边，排除mid
        }
    }
    return -1;//没找到
}
```

2)寻找左边界（第一个>=target的位置）

```c++
int lowerBound(vector<int>&nums,int target){
    int l=0;
    int r=nums.size()-1;
    int ans=nums.size();// 初始化为数组长度（表示没找到）
    while(l<=r){
        int mid=l+(r-l)/2;
        if(nums[mid]>=target){
            ans=mid;
            r=mid-1;// 继续向左找更小的
        }else{
            l=mid+1;
        }
    }
    return ans;
}
```

3)寻找右边界（最后一个<=target的位置）

```c++
int upperBound(vector<int>&nums,int target){
    int l=0;
    int r=nums.size()-1;
    int ans=nums.size();// 初始化为数组长度（表示没找到）
    while(l<=r){
        int mid=l+(r-l)/2;
        if(nums[mid]<=target){
            ans=mid;
            l=mid+1;// 继续向右找更大的
        }else{
            r=mid-1;
        }
    }
    return ans;
}
```

### 74.搜索二维矩阵

给你一个满足下述两条属性的 `m x n` 整数矩阵：

- 每行中的整数从左到右按非严格递增顺序排列。
- 每行的第一个整数大于前一行的最后一个整数。

给你一个整数 `target` ，如果 `target` 在矩阵中，返回 `true` ；否则，返回 `false` 。

**思路：**利用两次二分部查找实现。第一次二分查找用来寻找这个数所在的行数，第二次查找来寻找target在本行中的哪一个位置。

```c++
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int t) {
        //使用两次二分查找。
        int n=matrix.size();//行数
        int m=matrix[0].size();//列数
        //先处理空白
        if(matrix.empty()) return 0;
        //第一次二分查找，用来寻找比目标值所在的那一层（最大的首元素比t小的
        int top=0;
        int bottom=n-1;
        int tr=-1;//target所在行
        while(top<=bottom){
            int mid=top+(bottom-top)/2;
            if(matrix[mid][0]<=t){
                tr=mid;
                top=mid+1;
            }else{
                bottom=mid-1;
            }
        }
        if(tr==-1) return 0;
        //第二次二分查找，来找到底有无
        int l=0;
        int r=m-1;
        while(l<=r){
            int mid=l+(r-l)/2;
            if(matrix[tr][mid]==t){
                return 1;
            }else if(matrix[tr][mid]<t){
                l=mid+1;
            }else{
                r=mid-1;
            }
        }
        return 0;
    }
};
```

### 153.寻找排序数组中的最小值

已知长度为 `n` 的升序数组经 1 到 n 次旋转后得到输入数组。旋转一次指将最后一个元素移到最前。元素互不相同，请找出最小元素，要求 O(log n)。

- **思路**：旋转后是「部分有序」，最小值在「无序」那半。二分：若 `nums[mid] > nums[r]` 说明最小值在右半，`l = mid+1`；否则在左半（含 mid），`r = mid`。用 `while(l < r)`，最后 `nums[l]` 即为最小值。

- **代码**：

  ```c++
  class Solution {
  public:
      int findMin(vector<int>& nums) {
          int n = nums.size();
          int l = 0, r = n - 1;
          while (l < r) {
              int mid = (l + r) / 2;
              if (nums[mid] > nums[r])
                  l = mid + 1;
              else
                  r = mid;
          }
          return nums[l];
      }
  };
  ```

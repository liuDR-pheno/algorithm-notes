# 算法刷题笔记

这里记录自己刷算法题的心得和思路。

## 说明

题解代码存放在项目根目录的 **`code/`** 文件夹中，包含：

- **链表** - 翻转链表、倒数第 k 个、合并有序链表等
- **数组** - 两数之和、三数之和、最大子序和、数组扁平化等
- **字符串** - 最长回文子串、有效括号、解码等
- **树与图** - 二叉树深度、反转二叉树、岛屿面积等
- **动态规划** - 爬楼梯、编辑距离、不同路径等
- **手写题** - bind、Promise、发布订阅、深拷贝等
- **其他** - 二分查找、螺旋矩阵、接雨水等

本地查看题解：在仓库中打开 `code/` 目录即可看到所有 `.js` 题解文件。

后续可以在这里用 Markdown 补充每道题的思路笔记，或链接到具体题解文件。



# 排序

## 堆排序

### 算法思路

重点：两个函数

把完全二叉树进行与数组的映射，自己算清楚下标的换算，然后从二叉树的最后一个叶子结点的父节点进行调整堆（int i=n/2-1），到这一步已经完成建堆，此时堆顶是数组的最大值，接着把堆顶和最后一个数交换，把最大值放在数组最后，然后踢出我们需要排序的数组中，因为此时已经把堆顶换成小数了，我们需要对这个堆重新进行调整，取出新的堆顶（max），重复循环这个过程，这样，我们就将这个数组从小到大排序好了。

1）heapify：进行调整大根堆

```c++
//堆化（大根堆，堆顶是max）
//n->堆大小，i->要调整的结点下标
void heapify(vector<int>& a,int n,int i){
            int largest=i;//假设父节点是最大值
            int left=2*i+1;
            int right=2*i+2;
            if(left<n&&a[left]>a[largest]){
                largest=left;
            }
            if(right<n&&a[right]>a[largest]){
                largest=right;
            }
            if(largest!=i){
                swap(a[i],a[largest]);
                heapify(a,n,largest);//重新调整被影响的堆
            }
        }
//建立大根堆，从后往前调整
for(int i=n/2-1;i>=0;i--){
    heapify(a,n,i);
}
```

2）排序

```c++
for(int i=n-1;i>0;i--){
                swap(a[0],a[i]);//最大值放到末尾
                heapify(a,i,0);
```

### 135.分发糖果

`n` 个孩子站成一排，整数数组 `ratings` 表示每个孩子的评分。每个孩子至少 1 个糖果，相邻两个孩子中评分更高的获得更多糖果。求最少糖果数目。

- **思路**：两次遍历。从左往右：若 `ratings[i] > ratings[i-1]` 则 `l[i] = l[i-1]+1`；从右往左：若 `ratings[i] > ratings[i+1]` 则 `r[i] = r[i+1]+1`。每个位置取 `max(l[i], r[i])` 累加。

- **代码**：

  ```c++
  class Solution {
  public:
      int candy(vector<int>& ra) {
          int n = ra.size();
          if (n == 0) return 0;
          vector<int> l(n, 1), r(n, 1);
          for (int i = 1; i < n; i++)
              if (ra[i] > ra[i-1]) l[i] = l[i-1] + 1;
          for (int i = n - 2; i >= 0; i--)
              if (ra[i] > ra[i+1]) r[i] = r[i+1] + 1;
          int ans = 0;
          for (int i = 0; i < n; i++) ans += max(l[i], r[i]);
          return ans;
      }
  };
  ```

- 题解：`code/135.分发糖果.cpp`

### 136.只出现一次的数字

非空整数数组 `nums`，除某个元素只出现一次外，其余均出现两次。找出只出现一次的元素。要求线性时间、常量额外空间。

- **思路**：解法一：哈希表统计出现次数，空间 O(n)。解法二（推荐）：**异或**。`a ^ a = 0`，`a ^ 0 = a`，所有数异或后结果即为只出现一次的数，时间 O(n)，空间 O(1)。

- **代码**（哈希）：

  ```c++
  class Solution {
  public:
      int singleNumber(vector<int>& nums) {
          unordered_map<int,int> mp;
          for (int x : nums) mp[x]++;
          for (auto& val : mp)
              if (val.second == 1) return val.first;
          return -1;
      }
  };
  ```

- **代码**（异或 O(1) 空间）：

  ```c++
  int singleNumber(vector<int>& nums) {
      int ans = 0;
      for (int x : nums) ans ^= x;
      return ans;
  }
  ```

- 题解：`code/136.只出现一次的数字.cpp`

### 204.计数质数

给定整数 `n`，返回所有小于非负整数 `n` 的质数的数量。

- **思路**：**埃氏筛**。用 `isprime[i]` 表示 `i` 是否为质数，从 2 开始，若 `i` 是质数则把其所有倍数标记为非质数。只需枚举到 `i*i <= n`（或 `i <= sqrt(n)`），因为更大因子的倍数已在前面被标记。最后统计 `[2, n-1]` 中 `isprime` 为真的个数。

- **代码**：

  ```c++
  class Solution {
  public:
      int countPrimes(int n) {
          if (n <= 2) return 0;
          vector<bool> isprime(n + 1, true);
          isprime[0] = isprime[1] = false;
          for (int i = 2; i * i < n; i++) {
              if (!isprime[i]) continue;
              for (int j = i * i; j < n; j += i)
                  isprime[j] = false;
          }
          int ans = 0;
          for (int i = 2; i < n; i++)
              if (isprime[i]) ans++;
          return ans;
      }
  };
  ```

- 题解：`code/204.计数质数.cpp`
